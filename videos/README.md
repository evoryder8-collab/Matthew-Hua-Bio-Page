# /videos — media for the cinematic archive

Drop optimized clips here. The gallery in index.html references paths such as videos/hero-loop.mp4 and videos/awards/european-championship-2024.mp4 — each provided as both .webm and .mp4.

Expected structure:
videos/hero-loop.webm + .mp4   — short silent looping hero film
videos/awards/    — championship and award clips
videos/stage/     — judging and awarding-on-stage moments
videos/practice/  — bodywork detail
videos/posters/   — optional still frames

See the root README for the full structure and ffmpeg encoding commands. Until real files are added, the gallery shows elegant gradient posters.
